## Usage

Once installed, use the `/r-pkg` command in Antigravity to access the R Package Manager menu.

### Commands
- `List Installed Packages`: View all packages in your library.
- `Check Dependencies`: Scan your current workspace for missing R libraries.
- `Create New Package`: Scaffold a new R package from scratch.
- `Run Health Check`: Diagnose your R environment.

## Requirements

- R (version 4.0 or higher) must be installed on your system.
- For Windows users, ensuring R is in your system PATH is recommended.
